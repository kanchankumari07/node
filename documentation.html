Movie API

API Endpoints

URL HTTP Method Request body data format Response body data format
Allow existing users log in /login POST A JSON object holding data about the user including the their password
Get a list of all movies /movies GET A JSON object holding data about all the movies
Get a single movie by title /movies/:title GET A JSON object holding data about a single movie
Get Genre by Name /movies/genre/:Name GET A JSON object holding data about the genre of a movie
Get director data by name /movies/director/:Name GET A JSON object holding data about the data of a director
Get all Users /users GET A JSON object holding data of all users
Get a User by Username /users/:Username GET A JSON object holding data of a specific user
Delete movie from favorite list /users/:Username/movies/:MovieID DELETE A JSON Object holding data about the User and the movie that was removed
